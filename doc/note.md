## gitの初期設定

git config --global user.email メールアドレス
登録一覧の確認
git config --list

cat ~/.gitconfig

## よく使用するコマンド
cdコマンド
ディレクトリを移動します。

lsコマンド
ディレクトリの内容を表示します。 ls -a コマンドで、隠しファイルを含めたディレクトリ全内容を表示します。

mkdirコマンド
ディレクトリを新規作成します。

rmコマンド
ファイルを削除します。

cpコマンド
ファイルをコピーします。

mvコマンド
ファイルの移動とファイル名の変更を行います。

catコマンド
ファイルの中身を表示します。
